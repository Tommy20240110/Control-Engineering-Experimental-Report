# 数字图像处理实验报告

## 项目简介

本项目为华南师范大学（SCNU）数字图像处理课程实验报告，使用 LaTeX 排版，基于自定义 `SCNU_Experimental_Report` 模板。

## 编译方法

1. 打开 `Ex*/20233902058Ex*.tex`
2. 在 `LaTeX Workshop` 拓展 → 选择 `latexmk (xelatex)`

编译后 PDF 生成在 `.build/` 文件夹下。

## 注意事项

- 确保 `common/fonts/` 下有 `FangSong_GB2312.ttf`、`KaiTi_GB2312.ttf`、`SimHei.ttf`
- 不需要额外配置 VS Code
